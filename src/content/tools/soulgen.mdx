## О сервисе

SoulGen — специализированный генератор изображений, фокусирующийся на создании портретов. Сервис предлагает два стиля: фотореалистичный и аниме.

## Особенности

- **Специализация** — оптимизирован для портретов
- **Два стиля** — реализм и аниме
- **Простота** — интуитивный интерфейс
- **Россия** — принимает карты РФ

## Возможности

- Генерация по текстовому описанию
- Генерация по референсному фото
- Редактирование существующих изображений
- Outpainting — расширение изображения
- Удаление одежды (Premium)

## Качество

SoulGen использует fine-tuned модели Stable Diffusion:
- Высокая детализация лиц
- Реалистичное освещение
- Качественная анатомия

## Как использовать

1. Выберите стиль (Real/Anime)
2. Опишите желаемый результат
3. Опционально загрузите референс
4. Нажмите Generate
5. Скачайте результат

## Советы

- Описывайте детали внешности
- Указывайте освещение и фон
- Используйте референсы для лучшего контроля
