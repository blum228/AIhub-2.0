## О платформе

Janitor AI — уникальная платформа, которая позволяет использовать различные AI-модели для общения с персонажами. Вы подключаете свой API-ключ и получаете полный контроль над качеством и стоимостью.

## Как это работает

1. Регистрируетесь на Janitor AI
2. Получаете API-ключ (OpenAI, Claude, или альтернативы)
3. Подключаете ключ в настройках
4. Общаетесь с персонажами

## Преимущества

- **Гибкость** — выбирайте любую AI-модель
- **Контроль** — вы платите только за использование
- **Качество** — используйте лучшие модели (GPT-4, Claude)
- **Без цензуры** — нет встроенных фильтров платформы

## Поддерживаемые API

| Провайдер | Модели | Цена примерно |
|-----------|--------|---------------|
| OpenAI | GPT-4, GPT-3.5 | $0.01-0.03/1K токенов |
| Anthropic | Claude 3 | $0.01-0.02/1K токенов |
| OpenRouter | Разные | Варьируется |
| Kobold | Локальные | Бесплатно |

## Встроенная модель

Janitor LLM — бесплатная встроенная модель:
- Не требует API-ключа
- Качество ниже GPT/Claude
- Подходит для тестирования

## Советы

- Начните с GPT-3.5 — дёшево и качественно
- Используйте OpenRouter для доступа к разным моделям
- Настройте jailbreak-промпт для лучших результатов
